# AI Job Market Trends & Skills Demand Analysis

## Project Overview
This project analyzes AI job market trends, salary patterns, and in-demand skills using SQL and Power BI.

## Tools Used
- SQL
- Power BI

## Key Analysis
- Identified the most in-demand AI skills
- Analyzed salary trends across job roles
- Compared remote and onsite opportunities
- Evaluated hiring trends across industries

## Dashboard Features
- KPI cards
- Salary trend analysis
- Skills demand visualization
- Job role comparison
- Interactive filters

## Files Included
- Dataset
- SQL Queries
- Power BI Dashboard
